dd-transfer-to-vault-cli
=========================

Provides the `transfer` command to interact with `dd-transfer-to-vault` service via its [REST API]{:target=_blank}.

SYNOPSIS
--------

```bash
transfer -p <pipeline> flush-work-to-vault
transfer -p <pipeline> status [ -a | --all-batches ]
```

For more information on a subcommand use:

```bash
transfer <subcommand> --help
```

[REST API]: {{ transfer_to_vault_api }}
